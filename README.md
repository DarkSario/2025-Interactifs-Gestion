# 2025-Interactifs-Gestion

Projet de support pour les interactifs de gestion (année 2025).

## Description

Ce dépôt contient les exercices, démonstrations et code associés aux interactifs de gestion.

## Installation

Recommandé : créer un environnement virtuel.

```bash
python -m venv .venv
source .venv/bin/activate   # macOS / Linux
.\.venv\Scripts\activate    # Windows
pip install -r requirements.txt
```
